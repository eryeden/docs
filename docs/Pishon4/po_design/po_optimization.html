Covariance formula#

自己相関の計算をする。以下が成り立つとする。

 aσp2= aCovp( ap^, ap^)\ _a\sigma^2_{p}=\ _ aCov_p(\ _a\hat{p}, \ _a\hat{p}) a​σp2​= a​Covp​( a​p^​, a​p^​)
 aσr,i2= aCovr( ar^i, ar^i)\ _a\sigma^2_{r,i}=\ _ aCov_r(\ _a\hat{r}_i, \ _a\hat{r}_i) a​σr,i2​= a​Covr​( a​r^i​, a​r^i​)
 aσr2= aCovr( ar^, ar^)\ _a\sigma^2_{r}=\ _ aCov_r(\ _a\hat{r}, \ _a\hat{r}) a​σr2​= a​Covr​( a​r^, a​r^)

すると、分散×定数の公式から以下が成り立つ。どっちも同じ式になる。

 aσr,i2=(1− acS ap(1+ acB))2 aσp2\ _a\sigma^2_{r,i}=\left(\frac{1-\ _ac_S}{\ _ap(1+\ _ac_B)}\right)^2 \ _a\sigma^2_{p} a​σr,i2​=( a​p(1+ a​cB​)1− a​cS​​)2 a​σp2​
 aσr2=(1− acS ap(1+ acB))2 aσp2\ _a\sigma^2_{r}=\left(\frac{1-\ _ac_S}{\ _ap(1+\ _ac_B)}\right)^2 \ _a\sigma^2_{p} a​σr2​=( a​p(1+ a​cB​)1− a​cS​​)2 a​σp2​

Phase1としては、これで良い？おそらくIntra assetの共分散と、inter assetの共分散は異なってくる。

Intra asset covariance#

以下が成り立つとする。

 aσr,i,j2= aCovr( ar^i, ar^j)\ _a\sigma^2_{r,i,j}=\ _ aCov_r(\ _a\hat{r}_i, \ _a\hat{r}_j) a​σr,i,j2​= a​Covr​( a​r^i​, a​r^j​)
 aσr,0,i2= aCovr( ar^, ar^i)\ _a\sigma^2_{r,0,i}=\ _ aCov_r(\ _a\hat{r}, \ _a\hat{r}_i) a​σr,0,i2​= a​Covr​( a​r^, a​r^i​)

共分散の計算:

Cov(X,Y)=E(XY)−E(X)E(Y)Cov(X, Y) = E(XY) - E(X)E(Y)Cov(X,Y)=E(XY)−E(X)E(Y)
 ar^i ar^i=(1− acS)2 ap2(1+ acB)2 ap^2−(1− acS)( api(1+ aciB)+ apj(1+ acjB)) ap2(1+ acB) ap^− api apj(1+ aciB)(1+ acjB) ap2(1+ acB)2\ _a\hat{r}_i \ _a\hat{r}_i = \frac{(1-\ _ac_S)^2}{\ _ap^2(1+\ _ac_B)^2}\ _a\hat{p}^2\\ - \frac{(1-\ _ac_S)(\ _ap_i(1+\ _ac_{iB})+\ _ap_j(1+\ _ac_{jB}))}{\ _ap^2(1+\ _ac_B)}\ _a\hat{p}\\ -\frac{\ _ap_i\ _ap_j(1+\ _ac_{iB})(1+\ _ac_{jB})}{\ _ap^2(1+\ _ac_B)^2} a​r^i​ a​r^i​= a​p2(1+ a​cB​)2(1− a​cS​)2​ a​p^​2− a​p2(1+ a​cB​)(1− a​cS​)( a​pi​(1+ a​ciB​)+ a​pj​(1+ a​cjB​))​ a​p^​− a​p2(1+ a​cB​)2 a​pi​ a​pj​(1+ a​ciB​)(1+ a​cjB​)​

であり、E(X2)=E(X)2+V(X)E(X^2)=E(X)^2+V(X)E(X2)=E(X)2+V(X)なので、

 aσr,i,j2= aσr,0,i2=(1− acS)2 ap2(1+ acB)2 aσp2\ _a\sigma^2_{r,i,j}=\ _a\sigma^2_{r,0,i}\\ =\frac{(1-\ _ac_S)^2}{\ _ap^2(1+\ _ac_B)^2}\ _a\sigma_p^2 a​σr,i,j2​= a​σr,0,i2​= a​p2(1+ a​cB​)2(1− a​cS​)2​ a​σp2​

になる。

Inter asset covariance Cov( ar^i, br^j)Cov(\ _a\hat{r}_i, \ _b\hat{r}_j)Cov( a​r^i​, b​r^j​)#

Cov( ar^i, br^j)=E( ar^i br^j)−E( ar^i)E( br^j)=(1− acS)(1− bcS) ap bp(1+ acB)(1+ bCB)(E( ap^ bp^)−E( ap^)E( bp^))Cov(\ _a\hat{r}_i, \ _b\hat{r}_j) = E(\ _a\hat{r}_i \ _b\hat{r}_j)-E(\ _a\hat{r}_i) E(\ _b\hat{r}_j)\\ =\frac{(1-\ _ac_S)(1-\ _bc_S)}{\ _ap\ _bp(1+\ _ac_B)(1+\ _bC_B)} (E(\ _a\hat{p} \ _b\hat{p}) - E(\ _a\hat{p})E(\ _b\hat{p}))Cov( a​r^i​, b​r^j​)=E( a​r^i​ b​r^j​)−E( a​r^i​)E( b​r^j​)= a​p b​p(1+ a​cB​)(1+ b​CB​)(1− a​cS​)(1− b​cS​)​(E( a​p^​ b​p^​)−E( a​p^​)E( b​p^​))

Cov( ap^, bp^)= abσp2Cov(\ _a\hat{p}, \ _b\hat{p}) = \ _{ab}\sigma_p^2Cov( a​p^​, b​p^​)= ab​σp2​
